Burada bulunan sql kütüphanesini indirip kendi managament studio programınızda databes içerisine import edin. Bilgiler ile bağlantısı tablolar üzerinden sağlandı.

Veritabanı Geri Yükleme (Restore):

SSMS'de "Object Explorer" penceresinden veritabanını seçin.
"Tasks" seçeneğine gelin ve "Restore" seçeneğini seçin.
"Database" bölümünde "From device" seçeneğini seçin ve gerekli yedekleme dosyasını seçin.
Gerekli diğer ayarları yapın ve "OK" düğmesine tıklayarak geri yüklemeyi başlatın.
